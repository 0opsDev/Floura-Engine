Farquhar is a Open Source c++ game engine created for the prepose of learning opengl

Features:
. glTF Model Loading
. Vertex and Fragment Shader loading
. Settings File
. Map Files with Map Settings
. ImGui Implementation
. OpenGl API
. Diffuse and Specular Texture Support
